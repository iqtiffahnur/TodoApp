# ✅ To Do App

A clean and focused To‑Do application 
This project is intentionally lightweight, easy to run, and structured to demonstrate code clarity and practical UI/UX.

---

## ✨ Features

- Add, edit, complete, and delete tasks
- Filter by status (All / Active / Completed)
- Persistent data (LocalStorage by default)
- Responsive layout and accessible controls
- Clean, readable component structure

### Prerequisites
- **Node.js** ≥ 18.x and **npm** ≥ 9.x  
  Check versions:
  ```bash
  node -v
  npm -v

Run locally
# 1) Clone
git clone git@github.com:iqtiffahnur/ToDoApp.git
cd ToDoApp

# 2) Install dependencies
npm install

# 3) Start dev server
npm start
# or, if using Vite:
# npm run dev

Open: http://localhost:3000 (or the port shown in your terminal)
